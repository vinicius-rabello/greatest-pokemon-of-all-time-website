Who is the greatest Pokémon of All Time?

This website aims to answer this question! This table compiles data from all recorded VGC tournaments, sourced from Limitless VGC. It introduces a quantitative metric (see here) that calculates how great a Pokémon is based on the entire history of competitive play. Explore the table below to see how your favorites rank!

About the Greatness Metric: This metric takes the number of tournament wins, the importance of each event, and for how long each Pokémon has been great competitively.

Formula: (Number of Events) x (Placement Points) x (Event Multiplier) x (1 + Longevity/10)
Placement Points: Winner: 7pts, Runner-up: 4pts and Top-8: 1pt. Event Multiplier: Worlds: 5x, Internation: 3x, National: 2x and Others: 1x Longevity Bonus: For how many different generations a Pokémon got at least one Top-8 finish in any tournament.
